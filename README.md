# 🚀 Task Manager App

A full-stack Task Management Web Application built using React, Flask, and MongoDB.

## ✨ Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Search functionality
- Clean UI

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Flask (Python)
- Database: MongoDB

## ⚙️ Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd task-frontend
npm install
npm start
